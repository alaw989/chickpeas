  <files>components/sections/HomeInfo.vue</files>
  <action>
Add v-motion directives to animate the info section content on scroll:

1. Add fade-in + slide-up to the "Our Story" card (first white card):
```html
<div
  class="bg-white rounded-2xl shadow-lg p-8 md:p-10 w-full lg:w-1/2 text-left"
  v-motion
  :initial="{ opacity: 0, y: 24 }"
  :visible-once="{ opacity: 1, y: 0, transition: { duration: 500, delay: 100 } }"
>
```

2. Add fade-in + slide-up to the right container (image + info card):
```html
<div
  class="flex flex-col right-container w-full lg:w-1/2"
  v-motion
  :initial="{ opacity: 0, y: 24 }"
  :visible-once="{ opacity: 1, y: 0, transition: { duration: 500, delay: 200 } }"
>
```

3. Animation parameters:
   - Initial state: `opacity: 0, y: 24` (invisible, 24px below final position)
   - Visible state: `opacity: 1, y: 0` (fully visible, in place)
   - Duration: 500ms (0.5s - subtle, not slow)
   - Delay: staggered (100ms for left, 200ms for right)

4. Using `visible-once` (not `visible`) so animation only plays once, not on every scroll in/out.

Note: The v-motion directive is auto-imported from @vueuse/motion/nuxt (configured in Phase 1).
